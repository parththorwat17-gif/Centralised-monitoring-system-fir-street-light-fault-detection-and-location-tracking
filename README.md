# Smart Street Light Monitoring System

## Overview

The Smart Street Light Monitoring System is designed to improve the maintenance and management of streetlights. It automatically detects faulty streetlights and reports issues in real time, reducing the need for manual inspections and ensuring efficient operation.

## Features

* Real-time streetlight status monitoring
* Automatic fault detection
* Reduced maintenance costs
* Faster issue reporting
* Improved public safety
* User-friendly dashboard (optional)

## Problem Statement

Traditional streetlight maintenance relies on manual inspections, which are time-consuming, inefficient, and costly. This project aims to automate fault detection and monitoring to improve efficiency and reliability.

## Technologies Used

* Python
* IoT Sensors
* Arduino/ESP32
* MQTT/HTTP Protocol
* Database (MySQL/Firebase)
* Web Dashboard

## System Architecture

1. Sensors collect streetlight status data.
2. Data is transmitted to the server.
3. The system analyzes the data for faults.
4. Fault alerts are generated and displayed on the dashboard.
5. Maintenance teams receive notifications.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/smart-street-light-monitoring.git
   ```
2. Navigate to the project directory:

   ```bash
   cd smart-street-light-monitoring
   ```
3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:

   ```bash
   python app.py
   ```
